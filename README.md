# jannismoore/skills — ai agent skills for claude code, cursor & more

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Reusable AI agent skills for [Claude Code](https://docs.anthropic.com/en/docs/agents-and-tools/claude-code/overview), [Cursor](https://www.cursor.com/), and other AI coding agents — built to be installed via the [skills.sh](https://skills.sh) ecosystem.

Each skill encodes real-world workflows, patterns, and domain expertise into portable capabilities that any compatible AI agent can use. Practical, opinionated, and immediately useful.

Built by **[Jannis Moore](https://www.linkedin.com/in/jannismoore)** — entrepreneur and AI operator building systems that create leverage through AI. Founder of [Glaido](https://glaido.com), [Integraticus](https://integraticus.com), and [Voice AI Bootcamp](https://voiceaibootcamp.com).

## Installation

### Option 1: Install All Skills

```bash
npx skills add jannismoore/skills
```

### Option 2: Install Specific Skills

```bash
npx skills add jannismoore/skills@<skill-name>
```

### Manual Installation

Copy to your skills directory:

```bash
# Claude Code (global)
cp -r <skill-name> ~/.claude/skills/

# Claude Code (project-level)
cp -r <skill-name> .claude/skills/

# Cursor
cp -r <skill-name> .cursor/skills/
```
